# 习题集 · 即测即评

移动端答题页面，支持手机浏览器直接打开。两个习题集已合并为同一入口。

## 🎮 冲塔跑酷（轻量版）

**https://root-hz.github.io/work-analysis-quiz/game/**

- 竖屏一键跳跃，自动向前奔跑
- 越过石头、躲开飞鸟，跑满 1000m 通关
- 通关显示：**AAA你黄哥最牛**
- 本地记录最高距离（BEST）

本地试玩：启动服务器后访问 `http://<IP>:8080/game/`

## 在线访问（习题集）

**https://root-hz.github.io/work-analysis-quiz/**

打开后选择习题集即可开始答题：

- 工作分析（100 题）
- 马克思主义基本原理（262 题）

旧链接 `.../marxism/` 会自动跳转到马克思主义习题集。

（首次部署后可能需要 1-3 分钟生效）

## 本地运行

```powershell
python -m http.server 8080 --bind 0.0.0.0
```

手机与电脑同一 WiFi 下访问 `http://<电脑IP>:8080/`

## 功能说明

- 首页选择习题集，点击进入答题
- 答错即时显示正确答案，答对自动进入下一题
- 无时间限制
- 全部完成后显示得分，并汇总全部错题
